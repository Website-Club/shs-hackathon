## Getting Started <a name = "getting-started"></a>

### Installing

1. Clone the repository

```bash
git clone https://github.com/Website-Club/shs-hackathon.git
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
